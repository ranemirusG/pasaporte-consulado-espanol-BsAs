Fechas de apertura de citas previas

Las citas previas para los diferentes servicios de este Consulado General se obtienen en este sitio web. La gestión de todos los trámites se realizará exclusivamente con cita previa. En caso de cancelación de las citas por parte de este Consulado General, la reasignación de la cita para un día y hora alternativo se realizará por etapas respetando el orden cronológico de las citas obtenidas.

Este Consulado General concede mensualmente, en función de las capacidades disponibles, un cupo de turnos de cita previa, para fechas más cercanas, de la agenda del servicio de pasaportes. Consulte este sitio web diariamente para conocer con antelación el día y la hora en el que se habilitarán los turnos.
